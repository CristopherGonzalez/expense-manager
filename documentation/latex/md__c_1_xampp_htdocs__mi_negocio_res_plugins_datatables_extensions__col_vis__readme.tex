Col\+Vis adds a button to the toolbars around Data\+Tables which gives the end user of the table the ability to dynamically change the visibility of the columns in the table\+:


\begin{DoxyItemize}
\item Dynamically show and hide columns in a table
\item Very easy integration with Data\+Tables
\item Ability to exclude columns from being either hidden or shown
\item Save saving integration with Data\+Tables
