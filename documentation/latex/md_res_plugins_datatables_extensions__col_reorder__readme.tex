Col\+Reorder adds the ability for the end user to click and drag column headers to reorder a table as they see fit, to Data\+Tables. Key features include\+:


\begin{DoxyItemize}
\item Very easy integration with Data\+Tables
\item Tight integration with all other Data\+Tables plug-\/ins
\item The ability to exclude the first (or more) column from being movable
\item Predefine a column order
\item Save staving integration with Data\+Tables
\end{DoxyItemize}\hypertarget{md_res_plugins_datatables_extensions__col_reorder__readme_autotoc_md64}{}\doxysection{Installation}\label{md_res_plugins_datatables_extensions__col_reorder__readme_autotoc_md64}

This will allow the pages in the examples to operate correctly.
