C\+K\+Editor is a text editor to be used inside web pages. It\textquotesingle{}s not a replacement for desktop text editors like Word or Open\+Office, but a component to be used as part of web applications and websites.\hypertarget{md_res_plugins_ckeditor__r_e_a_d_m_e_autotoc_md57}{}\doxysection{Documentation}\label{md_res_plugins_ckeditor__r_e_a_d_m_e_autotoc_md57}
The full editor documentation is available online at the following address\+: \href{http://docs.ckeditor.com}{\texttt{ http\+://docs.\+ckeditor.\+com}}\hypertarget{md_res_plugins_ckeditor__r_e_a_d_m_e_autotoc_md58}{}\doxysection{Installation}\label{md_res_plugins_ckeditor__r_e_a_d_m_e_autotoc_md58}
Installing C\+K\+Editor is an easy task. Just follow these simple steps\+:


\begin{DoxyEnumerate}
\item {\bfseries{Download}} the latest version from the C\+K\+Editor website\+: \href{http://ckeditor.com}{\texttt{ http\+://ckeditor.\+com}}. You should have already completed this step, but be sure you have the very latest version.
\item {\bfseries{Extract}} (decompress) the downloaded file into the root of your website.
\end{DoxyEnumerate}

{\bfseries{Note\+:}} C\+K\+Editor is by default installed in the {\ttfamily ckeditor} folder. You can place the files in whichever you want though.\hypertarget{md_res_plugins_ckeditor__r_e_a_d_m_e_autotoc_md59}{}\doxysection{Checking Your Installation}\label{md_res_plugins_ckeditor__r_e_a_d_m_e_autotoc_md59}
