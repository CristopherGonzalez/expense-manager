Table\+Tools is a plug-\/in for the Data\+Tables H\+T\+ML table enhancer, which adds a highly customisable button toolbar to a Data\+Table. Key features include\+:


\begin{DoxyItemize}
\item Copy to clipboard
\item Save table data as C\+SV, X\+LS or P\+DF files
\item Print view for clean printing
\item Row selection options
\item Easy use predefined buttons
\item Simple customisation of buttons
\item Well defined A\+PI for advanced control
\end{DoxyItemize}\hypertarget{md__c_1_xampp_htdocs__mi_negocio_res_plugins_datatables_extensions__table_tools__readme_autotoc_md81}{}\doxysection{Installation}\label{md__c_1_xampp_htdocs__mi_negocio_res_plugins_datatables_extensions__table_tools__readme_autotoc_md81}

This will allow the pages in the examples to operate correctly.
