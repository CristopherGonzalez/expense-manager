When making use of Data\+Tables\textquotesingle{} x-\/axis scrolling feature ({\ttfamily scrollX}), you may wish to fix the left or right most columns in place. This plug-\/in for Data\+Tables provides exactly this option (for non-\/scrolling tables, please use the Fixed\+Header plug-\/in, which can fix headers, footers and columns). Key features include\+:


\begin{DoxyItemize}
\item Freezes the left most column to the side of the table
\item Option to freeze two or more columns
\item Full integration with Data\+Tables\textquotesingle{} scrolling options
\end{DoxyItemize}\hypertarget{md__c_1_xampp_htdocs__mi_negocio_res_plugins_datatables_extensions__fixed_columns__readme_autotoc_md76}{}\doxysection{Installation}\label{md__c_1_xampp_htdocs__mi_negocio_res_plugins_datatables_extensions__fixed_columns__readme_autotoc_md76}

This will allow the pages in the examples to operate correctly. To see the examples running, open the {\ttfamily examples} directory in your web-\/browser.\hypertarget{md__c_1_xampp_htdocs__mi_negocio_res_plugins_datatables_extensions__fixed_columns__readme_autotoc_md77}{}\doxysection{Basic usage}\label{md__c_1_xampp_htdocs__mi_negocio_res_plugins_datatables_extensions__fixed_columns__readme_autotoc_md77}
