{\ttfamily  Header/\+Footer Formatting Syntax taken from Office Open X\+ML Part 4 -\/ Markup Language Reference, page 1970\+:}

{\ttfamily There are a number of formatting codes that can be written inline with the actual header / footer text, which affect the formatting in the header or footer.}

{\ttfamily Example\+: This example shows the text \char`\"{}\+Center Bold Header\char`\"{} on the first line (center section), and the date on the second line (center section). \&C\+Center \&\char`\"{}-\/,\+Bold\char`\"{}Bold\&\char`\"{}-\/,\+Regular\char`\"{}Header\+\_\+x000\+A\+\_\+\&D}

{\ttfamily General Rules\+: There is no required order in which these codes must appear.}

{\ttfamily The first occurrence of the following codes turns the formatting ON, the second occurrence turns it O\+FF again\+:
\begin{DoxyItemize}
\item strikethrough
\item superscript
\item subscript Superscript and subscript cannot both be ON at same time. Whichever comes first wins and the other is ignored, while the first is ON. \&L -\/ code for \char`\"{}left section\char`\"{} (there are three header / footer locations, \char`\"{}left\char`\"{}, \char`\"{}center\char`\"{}, and \char`\"{}right\char`\"{}). When two or more occurrences of this section marker exist, the contents from all markers are concatenated, in the order of appearance, and placed into the left section. \&P -\/ code for \char`\"{}current page \#\char`\"{} \&N -\/ code for \char`\"{}total pages\char`\"{} \&font size -\/ code for \char`\"{}text font size\char`\"{}, where font size is a font size in points. \&K -\/ code for \char`\"{}text font color\char`\"{} R\+GB Color is specified as R\+R\+G\+G\+BB Theme Color is specifed as T\+T\+S\+NN where TT is the theme color Id, S is either \char`\"{}+\char`\"{} or \char`\"{}-\/\char`\"{} of the tint/shade value, NN is the tint/shade value. \&S -\/ code for \char`\"{}text strikethrough\char`\"{} on / off \&X -\/ code for \char`\"{}text super script\char`\"{} on / off \&Y -\/ code for \char`\"{}text subscript\char`\"{} on / off \&C -\/ code for \char`\"{}center section\char`\"{}. When two or more occurrences of this section marker exist, the contents from all markers are concatenated, in the order of appearance, and placed into the center section.
\end{DoxyItemize}}
